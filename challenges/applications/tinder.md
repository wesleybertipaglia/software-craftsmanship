# Software Craftsmanship - Tinder

## Introduction

Create a dating and social discovery platform where users can create profiles, swipe through matches, and connect with others. This challenge explores matchmaking algorithms, user profiles, and real-time interactions.

### Objectives

- Understand social discovery platform design
- Implement matching algorithms and user preferences
- Handle real-time messaging and notifications
- Learn about user privacy and safety features

### Instructions

1. **Environment Setup**:
    - Choose your preferred programming language and web framework
    - Set up a project structure following common conventions for your chosen language/framework (e.g., src/ or app/ for source files, tests/ for test files, assets/ for profile images)

2. **Implementation Details**:
    - Create user profiles with photos and preferences
    - Build swipe interface and matching algorithm
    - Implement chat functionality for matches
    - Add location-based discovery and filters
    - Handle user verification and safety features

3. **Testing**:
    - Test matching algorithm and user preferences
    - Verify chat functionality and real-time updates
    - Check location-based features
    - Test with multiple users and interactions

### Possible Improvements

- Add video profiles and calls
- Implement advanced matching with AI
- Add events and social features
- Create premium subscription features

## Conclusion

By completing this challenge, you will gain experience in building social discovery platforms and understanding user interaction systems.

Happy coding!
